## LAB 1 – FMG Setup and SD-WAN Overlay Template Creation:

This lab will review how to configure a blank factory reset FortiManager.  This will include some CLI settings and admin settings in FMG. Device Group creation and all of the steps necessary to fully run an SD-WAN Overlay Template (SOT) will also be covered.


## Objectives:

- Initial FortiManager Setup
  - Review steps that have already been completed to save you time
- Add FAZ to FMG
- Create Device Groups for Branches
- Add HUBs to FMG 
- Create SD-WAN Overlay Template (SOT)

## Time to Complete

Estimated: 30 Minutes
