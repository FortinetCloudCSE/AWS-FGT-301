In this lab we will be pushing the SOT generated Provisioning Templates and Policy Package to the Hub FGTs.  For the Branches, we will  create model devices, assign a Device Blueprint, and define their metadata variables via the .CSV import model device creation feature.

## Objectives:
 - Installing HUB1 and HUB2 provisioning templates and policy package
 - Create Branch Device Blueprint
 - Create .csv for Branch Model Device Creation
 - Create Branch Model devices via .CSV Import
 - Auto-link Branch FGTs
